# WorksheetGen

> AI-powered worksheet generator for primary and middle school teachers.

🔗 **Live demo:** *(deploy to Vercel and add URL)*

---

## About

WorksheetGen creates kid-friendly, print-ready worksheets for Classes 1–8 in seconds. Choose subject and topic, pick a length, and get engaging activities with cute illustrations, Indian-context word problems, and built-in answer keys.

---

## Features

- 🎨 50+ kid-friendly SVG illustrations (animals, fruits, shapes, numbers, etc.)
- 📚 8 subjects: Math, English, EVS, Hindi, Science, Social Science, GK, Computer
- 👶 Age-appropriate activity types (counting, matching, coloring for young; word problems, comprehension for older)
- 🇮🇳 Indian context (Indian names, ₹ currency, local scenarios)
- 🌐 Multilingual: English, Hindi, Hinglish, Marathi, Tamil, Bengali, Gujarati
- 📝 Built-in answer keys
- 🖨️ Print-ready (A4, beautiful spacing)

---

## Tech stack

- Frontend: Vanilla HTML / CSS / JavaScript
- Backend: Vercel Serverless Functions
- AI: Google Gemini 2.5 Flash
- Hosting: Vercel (free tier)

---

## Deployment

1. Fork this repo
2. Import to [Vercel](https://vercel.com/new)
3. Add `GEMINI_API_KEY` in Project Settings → Environment Variables
4. Deploy
